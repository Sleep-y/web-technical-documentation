CSS

CSS or Cascading Style Sheets is used to beautify websites such as giving the background a different colour or making elements of a website appear red.

CSS is also usually implemented externally via the < head > and < link rel > tags.

CSS utilizes classes and ids in order to implement a style on certain elements or a group of elements. CSS can also specify html tags to stylize all elements with that tag.
